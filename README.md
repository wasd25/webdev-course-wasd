# Código Fuente del Curso "Fundamentos de Desarrollo Web"

¡Bienvenido al repositorio de código de nuestro curso!

Aquí encontrarás todos los ejemplos de código HTML y CSS utilizados en las lecciones de video.

## Estructura de Carpetas

* **/starter-files**: Contiene el código "inicial" para algunas lecciones. Es el punto de partida ideal si quieres programar junto con el video.
* **/completed-examples**: Contiene el código "final" de cada lección. Úsalo como referencia si te atascas o si quieres ver el resultado terminado.
* **/scripts**: Contiene los guiones de narración detallados para cada video del curso.

## ¿Cómo usar esto?

1.  **Explora**: Navega a la carpeta de la lección que te interesa (ej. `completed-examples/lesson-05-profile-page/`).
2.  **Copia el código**: Puedes copiar el contenido de los archivos `index.html` y `style.css`.
3.  **Pruébalo tú mismo**: Pega el código en un editor en línea como [CodePen.io](https://codepen.io/) para experimentar y hacer tus propios cambios.

¡Gracias por aprender con nosotros!
