# Fundamentos de Desarrollo Web: ¡Crea tu Primera Página!

## Resumen del Curso
¡Bienvenido/a! [cite_start]Este curso de 1 hora te enseñará a crear sitios web sencillos desde cero[cite: 38]. Aprenderás los dos lenguajes básicos de la web: HTML (para la estructura) y CSS (para el estilo).

[cite_start]¡Y lo mejor de todo es que **no requiere descargas**! [cite: 41, 85] [cite_start]Usaremos herramientas 100% en línea y gratuitas[cite: 42].

* [cite_start]**Duración total**: ~60 minutos [cite: 87]
* [cite_start]**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación[cite: 37, 88].
* **Prerrequisitos**: Ninguno. [cite_start]Solo necesitas curiosidad y un navegador web[cite: 73, 89].
* **Herramientas necesarias**: ¡Solo tu navegador web! (Usaremos CodePen.io) [cite_start][cite: 70, 90].
* [cite_start]**Repositorio de código fuente**: [PEGA AQUÍ TU ENLACE AL REPOSITORIO DE GITHUB] [cite: 91]

## Secuencia de la lección

### Lección 1: ¿Qué es el Desarrollo Web? (5 minutos)
* **Descripción**: ¿Alguna vez te has preguntado cómo funcionan tus sitios web favoritos? [cite_start]En esta lección, exploraremos qué es un sitio web y por qué HTML y CSS son los bloques de construcción fundamentales[cite: 45, 94].
* [cite_start]**Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 1)] [cite: 95]
* [cite_start]**Consejos clave**: Piensa en HTML como los "huesos" (la estructura) y en CSS como la "ropa" (el estilo) de una página web[cite: 96].

### Lección 2: Introducción a HTML (10 minutos)
* **Descripción**: ¡Es hora de escribir código! Aprenderás las etiquetas HTML básicas y crearás tu primera página web en blanco. [cite_start]Veremos qué son las etiquetas, los elementos y los atributos[cite: 45, 99].
* [cite_start]**Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 2)] [cite: 99]
* [cite_start]**Consejos clave**: Las etiquetas más importantes son `<html>`, `<head>`, `<body>`, `<h1>` (para títulos) y `<p>` (para párrafos)[cite: 100].
* [cite_start]**Práctica**: [PEGA AQUÍ TU ENLACE DE CODEPEN (Lección 2)] (¡No necesitas registrarte!) [cite: 101]

### Lección 3: Añadir más elementos HTML (10 minutos)
* **Descripción**: Una página web es más que texto. [cite_start]Aprenderás a agregar listas para organizar información, imágenes (`<img>`) para hacerla visual y enlaces (`<a>`) para conectar con otras páginas[cite: 46, 104].
* [cite_start]**Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 3)] [cite: 105]
* **Consejos clave**: Usa `<ul>` para listas con viñetas e `<img>` para mostrar fotos. [cite_start]¡Recuerda el atributo `src` para la fuente de la imagen! [cite: 106]
* [cite_start]**Práctica**: [PEGA AQUÍ TU ENLACE DE CODEPEN (Lección 3)] (¡Comienza al instante!) [cite: 107]

### Lección 4: Introducción a CSS (15 minutos)
* **Descripción**: ¡Vamos a darle estilo! Es hora de aprender CSS. [cite_start]Veremos cómo usar selectores para "apuntar" a elementos HTML y cambiar sus propiedades como `color`, `font-size` (tamaño de letra) y `background-color` (color de fondo)[cite: 46, 109].
* [cite_start]**Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 4)] [cite: 110]
* **Consejos clave**: ¡Puedes cambiar el estilo de todas las etiquetas `<p>` a la vez! La sintaxis clave es: `selector { propiedad: valor; }`[cite: 111].
* [cite_start]**Práctica**: [PEGA AQUÍ TU ENLACE DE CODEPEN (Lección 4)] (¡Aplica estilo con un clic!) [cite: 112]

### Lección 5: Proyecto: Crear una Página de Perfil (15 minutos)
* **Descripción**: ¡Es hora de juntar todo! Usaremos todo lo aprendido de HTML y CSS para construir una página de perfil personal básica. [cite_start]Aprenderás a centrar contenido y a organizar tu página[cite: 47, 48, 114].
* [cite_start]**Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 5)] [cite: 115]
* [cite_start]**Consejos clave**: Usaremos propiedades de CSS como `text-align` y `border` (bordes) para que se vea genial[cite: 47, 116].
* [cite_start]**Proyecto Final**: [PEGA AQUÍ TU ENLACE DE CODEPEN (Lección 5)] (¡Crea tu Perfil!) [cite: 117]

### Lección 6: Consejos y Próximos Pasos (5 minutos)
* **Descripción**: ¡Felicidades! Has completado el curso. [cite_start]Hablaremos sobre los errores más comunes que cometen los principiantes y te daremos recursos para que sigas aprendiendo por tu cuenta[cite: 49, 119].
* [cite_start]**Enlace**: [PEGA AQUÍ TU ENLACE DE YOUTUBE (Video 6)] [cite: 120]
* **Consejos Clave**: ¡No te rindas! Revisa tu código buscando errores tipográficos. Un gran recurso para seguir es "MDN Web Docs".
