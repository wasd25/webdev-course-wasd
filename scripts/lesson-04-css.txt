Muy bien, nuestra página tiene estructura. Tiene huesos. Pero se ve... simple. Es hora de ponerle la ropa. ¡Es hora de CSS!

(Señalar la ventana de CSS)
Vamos a movernos a esta ventana.
La sintaxis de CSS, la forma en que funciona, es siempre la misma.
Es: `Selector { Propiedad: Valor; }`
(Repetir eso lentamente).
El **Selector** es A QUÉ le damos estilo (ej: al `<h1>`, al `<p>`).
La **Propiedad** es QUÉ queremos cambiar (ej: el `color`).
El **Valor** es CÓMO lo queremos cambiar (ej: a `blue`).

Vamos a probarlo. ¿Quieren cambiar TODA la página? El selector es `<body>`.
(Escribir en la ventana de CSS)
`body {`
`  background-color: #f0f0f0;`
`}`
(Explicar) "background-color es la propiedad. #f0f0f0 es el valor para un gris muy claro. ¡Y boom! El fondo cambió."

Ahora, quiero que mi título `<h1>` sea especial. El selector es `h1`.
(Escribir)
`h1 {`
`  color: blue;`
`  text-align: center;`
`}`
(Explicar) "Propiedad: `color`. Valor: `blue`. Propiedad: `text-align`. Valor: `center`. ¡Y miren! Nuestro título es azul y está centrado."

Ahora los párrafos. El selector es `p`.
(Escribir)
`p {`
`  font-size: 18px;`
`  font-family: Arial, sans-serif;`
`}`
(Explicar) "`font-size` es tamaño de letra. `18px` significa 18 píxeles. `font-family` es el tipo de letra. Le decimos 'usa Arial, y si no la tienes, usa cualquier fuente sans-serif'."

### Errores Comunes (¡El más importante!) (Extenso)
¡Escuchen! Esto es lo más importante de la lección. Aquí es donde el 99% de los principiantes se atascan.
**Error #1: Olvidar el punto y coma (`;`).**
(Ir al código CSS y **borrar intencionalmente el punto y coma** después de `18px`).
`p {`
`  font-size: 18px`
`  font-family: Arial, sans-serif;`
`}`

"¡Oh no! ¿Vieron eso? El tamaño de la letra (`font-size`) funcionó, ¡pero la fuente (`font-family`) no! El navegador se confundió. ¿Por qué?
Porque en CSS, cada regla, cada línea, *debe* terminar con un punto y coma (`;`). Es como el punto al final de una oración. Si no lo pones, el navegador junta las dos líneas y no entiende nada."
(Agregar el punto y coma de nuevo). "¡Ahí está! Vuelve a funcionar."

**Error #2: Olvidar una llave de cierre (`}`).**
(Borrar la llave de cierre `}` del selector `p`)
"¡Miren! ¡Ahora todo mi CSS se rompió! ¡Incluso el estilo del `<h1>` desapareció! ¿Por qué?
Porque al borrar la llave de cierre, el navegador piensa que todo lo que sigue (incluyendo el selector `img`) es parte del estilo de `p`. ¡Entró en pánico!"
(Poner la llave `}` de nuevo). "¡Perfecto!"

(Continuar con el guion)

`}`
