¡Es la hora del proyecto final! Vamos a usar todo lo que aprendimos para crear una bonita página de perfil[cite: 48].

(Empezar con el HTML del archivo /starter-files/lesson-05/index.html).
Ok, he preparado un HTML básico. Es casi todo lo que ya conocemos, pero notarán algo nuevo: `<div class="perfil-card">`.
Un `<div>` es solo un contenedor, una caja invisible. Y una "clase" (`class`) es un nombre especial que le damos para poder seleccionarlo fácilmente en CSS.

(Ir a la ventana de CSS)
¡Está todo desordenado! Vamos a arreglarlo.
Primero, el `body`. Usemos un truco de CSS moderno para centrar nuestra tarjeta de perfil perfectamente.
(Escribir)
`body {`
`  font-family: Arial, sans-serif;`
`  background-color: #f0f2f5;`
`  display: flex;`
`  justify-content: center;`
`  align-items: center;`
`  height: 100vh;`
`}`
"¡Boom! Centrado. `display: flex` es súper poderoso, lo usarán mucho."

Ahora, seleccionemos nuestra tarjeta. Como no es una etiqueta (como `p` o `h1`), sino una clase (`class`), no podemos solo escribir `perfil-card`. ¡Debemos usar un PUNTO (`.`) al inicio!"

(Escribir)
`.perfil-card {`
`  width: 300px;`
`  background-color: white;`
`  border-radius: 10px; /* Esquinas redondeadas */`
`  box-shadow: 0 4px 8px rgba(0,0,0,0.1); /* ¡Una sombra genial! */`
`  text-align: center; /* Centramos todo el texto */`
`  padding: 20px; /* Un poco de espacio interno */`
`}`

**ERRORES COMUNES (3-4 minutos)**
"¿Qué pasa si olvido el punto? (Borrar el punto de `.perfil-card`). ¡Todo desaparece! El navegador busca una etiqueta llamada `<perfil-card>`, que no existe. ¡El punto es obligatorio para las clases!"
(Poner el punto de nuevo).

(Continuación)
"¡Vamos a terminar esto! Hagamos la imagen redonda."
(Escribir)
`.perfil-card img {`
`  width: 150px;`
`  height: 150px;`

`}`
