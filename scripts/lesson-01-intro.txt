Seguro usas páginas como estas todos los días. ¿Pero alguna vez te has preguntado cómo funcionan? ¿Qué las hace visibles? ¿Qué las hace bonitas?

La respuesta es que todas, absolutamente todas, se construyen sobre dos lenguajes fundamentales: HTML y CSS.

(Mostrar un diagrama simple: HTML = Esqueleto, CSS = Ropa)
Quiero que piensen en esto de una forma muy simple.
Piensen en una página web como si fuera una persona.

**El HTML (HyperText Markup Language)** son los "huesos". Es el esqueleto.
El HTML le dice al navegador: "Oye, esto es un título", "esto es un párrafo de texto", "aquí va una imagen", "esto es un enlace". Le da la ESTRUCTURA. Sin HTML, no hay página web.

Pero claro, un esqueleto solo... es un poco aburrido, ¿verdad?

Ahí es donde entra el **CSS (Cascading Style Sheets)**.
El CSS es la "ropa", la "piel", el "estilo".
El CSS le dice al navegador: "Quiero que ese título sea azul y grande", "quiero que esa imagen tenga un borde redondeado" y "quiero que todo esté centrado y se vea genial". Le da el ESTILO.
