¡Bienvenidos de nuevo! Es hora de escribir nuestro primer código.

(Estar en CodePen.io, en una ventana nueva)
Estoy aquí en CodePen.io. Tenemos tres ventanas: HTML, CSS y JS. Por ahora, solo nos importa la de HTML.

(Empezar a escribir)
Toda página HTML necesita una estructura básica. Escribimos `<html>` y una etiqueta de cierre `</html>`. Casi todas las etiquetas de HTML tienen una de apertura y una de cierre con una diagonal.

Dentro de `<html>`, tenemos dos partes principales: la "cabeza" (`<head>`) y el "cuerpo" (`<body>`).
(Escribir las etiquetas <head> y <body>)
El `<head>` es para información *sobre* la página, como el título que ves en la pestaña del navegador. Por ejemplo: `<title>Mi Primera Página</title>`.

El `<body>` es todo lo que ves *dentro* de la página[cite: 100]. ¡Aquí es donde ocurre la diversión!

Vamos a añadir un título. Para eso, usamos la etiqueta `<h1>`, que significa "Heading 1", el título más importante.
(Escribir) `<h1>¡Hola Mundo!</h1>`
¡Miren! Al instante, aparece "Hola Mundo" en nuestra vista previa.

Ahora, un párrafo. Para eso, usamos la etiqueta `<p>`.
(Escribir) `<p>Estoy aprendiendo a programar mi primera página web.</p>`

**ERRORES COMUNES (3-4 minutos)**
Ahora, algo súper importante. ¿Qué pasa si me olvido de cerrar una etiqueta? Este es el error #1.
(Ir al código y **borrar intencionalmente la etiqueta de cierre `</p>`** de la primera línea de párrafo).

`<p>Estoy aprendiendo a programar...`
`<p>Me encanta jugar videojuegos.</p>`

"¡Miren! ¿Ven qué pasó? Ahora ambas líneas de texto están juntas, como si fueran un solo párrafo. El navegador no supo dónde terminaba el primer párrafo, así que simplemente siguió adelante. Por eso es VITAL que siempre cierren sus etiquetas. Voy a arreglarlo..."
(Volver a poner el `</p>`).

¡Listo! Ya creaste tu primera página web.
