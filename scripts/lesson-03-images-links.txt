Primero, ¿qué tal una lista de cosas que nos gustan? Usaremos una lista "no ordenada", que usa viñetas.
Usamos la etiqueta `<ul>` (Unordered List). Y dentro, cada elemento de la lista usa la etiqueta `<li>` (List Item)[cite: 106].
(Escribir)
`<h2>Mis Hobbies</h2>`
`<ul>`
  `<li>Jugar videojuegos</li>`
  `<li>Escuchar música</li>`
  `<li>Ver películas</li>`
`</ul>`
¡Genial!

¿Ya? ¡Perfecto! Yo agregaré...
(Escribir) `<li>Comer pizza</li>`
¡Fácil!

---
(Continuación)
Ahora, ¡enlaces! Los enlaces (`<a>`) son los que conectan la web. Usamos la etiqueta `<a>`, que significa "anchor" (ancla).
Esta etiqueta necesita un "atributo". Un atributo nos da más información. El atributo `href` le dice al navegador A DÓNDE ir[cite: 106].
(Escribir)
`<p>Visita mi sitio favorito: <a href="https://www.youtube.com">YouTube</a>.</p>`
¡Miren! Ahora "YouTube" es un enlace clickeable.

---
**ERRORES COMUNES (4-5 minutos)**
¿Qué pasa si escribo mal el atributo?
(Escribir) `<a ref="https://www.google.com">Google</a>`
"Miren, el texto aparece, pero no es un enlace. ¿Por qué? Porque escribí `ref` en lugar de `href`. El navegador no entendió la instrucción. ¡La ortografía es clave! Lo corrijo..."

Ahora, ¡imágenes! Usamos la etiqueta `<img>`. Esta etiqueta es especial, no tiene etiqueta de cierre.
PERO, necesita atributos OBLIGATORIOS[cite: 46]:
1. `src` (source, o fuente) para decirle dónde está la imagen.
2. `alt` (texto alternativo) para describir la imagen si no se puede cargar.

(Escribir)
`<img src="https://picsum.photos/id/237/400/300" alt="Un perro cachorro negro">`

"¡Ahí está! Un perro adorable. Ahora, ¿qué pasa si escribo mal el `src`?"
(Cambiar el `src` a `.../237/400/301` o una URL rota)
"¡Miren! La imagen se rompió. Pero, ¿ven ese texto que aparece? 'Un perro cachorro negro'. ¡Ese es nuestro texto `alt`! Por eso es tan importante."
(Arreglar el `src` de nuevo).
